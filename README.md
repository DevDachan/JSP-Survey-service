# Survey-service-JSP

기존에 존재하는 Google Forms과 Microsoft Forms의 서비스에서 사용자가 결과 페이지에 대한 프로그래밍적 요소를 추가할 수 있는 서비스입니다.
<br/>

## Getting Started

서비스는 기본적으로 JSP언어로 작성이 되었으며 tomcat container를 사용해 구동이 됩니다.



### 실행 조건

https://www.eclipse.org/downloads/

tomcat version 9.0
